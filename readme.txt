== Description ==

**WpCoordonnees** est un petit plugin permettant l'ajout des coordonn&eacute;es de votre entreprise ou autre dans votre site sans avoir &agrave; les mettre en dur dans votre code. Un menu donnera acc&egrave;s au formulaire afin de remplir les champs que vous souhaitez. Il existe 3 moyens de transmettre les coordonn&eacute;es. Par le code php, par le widget ou par le shortcode. Pour me contacter en priv&eacute; ou me faire un don, je vous invite &agrave; vous rendre sur <a href="http://mj-web.fr/">mon site</a>.

== Installation ==

Extraire le fichier zip et d&eacute;posez simplement le contenu dans le r&eacute;pertoire wp-content/plugins/ de votre installation de WordPress, puis activez le plugin depuis la page Plugins. Un menu WpCoordonnees apparaitra.
